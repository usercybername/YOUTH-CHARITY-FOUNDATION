# Youth Charity Foundation - NGO Management System

A comprehensive web-based management system for NGO project managers to manage volunteers, donations, beneficiaries, events, and generate impactful reports.

## Features

- 👤 **Volunteer Management** - Track volunteers, assignments, and hours
- 💰 **Donation & Fund Management** - Manage donations and track expenses
- 👥 **Beneficiary Management** - Track youth participants and their progress
- 📊 **Analytics & Reporting** - Generate impact reports and dashboards
- 📅 **Event & Program Management** - Organize activities and programs
- 📧 **Communication** - Send notifications and updates
- 🔐 **Secure Authentication** - Role-based access control

## Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **Authentication**: JWT
- **Deployment**: Docker ready

## Project Structure

```

```

## Getting Started

See individual README files in `/backend` and `/frontend` directories.

## Installation

### Prerequisites
- Node.js (v14+)
- PostgreSQL (v12+)
- Docker (optional)

### Quick Start

1. Clone the repository
2. Set up backend: `cd backend && npm install`
3. Set up frontend: `cd frontend && npm install`
4. Configure environment variables
5. Run migrations
6. Start development servers
